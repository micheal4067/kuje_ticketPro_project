Payment Scheme

The Payment Scheme page manages ticket pricing and additional fee settings for each entity. It ensures administrators and managers can set charges consistently across all entities.

Entity Selection

Administrators can view and update all entities, while managers only manage their assigned one. A dropdown menu (Select Entity) allows choosing the target database before applying changes.

Ticket Prices

The following categories are available for configuration:

  • Tricycle
  • Car
  • Small Truck
  • Large Truck
  • Ticket Loss

Each vehicle type has a dedicated input field for updating its base price.

Additional Fee Settings

An optional toggle enables applying extra charges based on time. When enabled, the following fields appear:

  • Apply Fee Every (Hours): Interval for repeated fee application.
  • Additional Fee Amount: The amount charged per interval.
  • Apply Fee After (Hours): Delay before extra charges begin.
