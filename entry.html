Entry Surveillance

The Entry Dashboard is the main interface operators use for daily activities, such as issuing tickets, monitoring parking slots, and managing entry or exit barriers.

Ticket Issuance

Different types of tickets are displayed as interactive buttons. Operators can quickly select a ticket type, including special options like VIP passes, to process vehicle entry smoothly. Ticket Issuance triggers corresponding barrier to open, giving access to the vehicle.

Parking Monitoring

The sidebar shows live updates of available parking slots, vehicle categories (cars, tricycles, trucks, VIP), and the total parking fees collected. This gives operators a clear view of space usage at all times.

Gate Control

Operators can directly open or manage entry and exit barriers through the dashboard using simple control buttons, ensuring smooth traffic flow.

Search & Logs

A search feature is provided to look up tickets by their numbers, while activity logs display recent operations for accountability and tracking.

Additional Tools

The dashboard also supports features like light/dark mode switching, auto-refresh, and mobile-friendly layouts, making it efficient to use across different devices.
